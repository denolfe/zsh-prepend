# zsh-prepend

ZSH plugin to prepend text to any file

## Usage

```sh
$ prepend "text to prepend" filename
```

## Installation

#### Using antibody:

```sh
$ antibody bundle denolfe/zsh-prepend
```

#### Using antigen:

```sh
$ antigen bundle denolfe/zsh-prepend
```

#### Using zgen:

```sh
$ zgen load denolfe/zsh-prepend
```
